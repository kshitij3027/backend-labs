# Log Shipping Client

A TCP client that reads logs from a local file and forwards them over the network to a centralized TCP log server. Supports two modes: **continuous** (watches a file for new lines) and **batch** (sends all existing lines and exits).

## Tech Stack

- Language: Python 3.12
- No external dependencies (stdlib only: `socket`, `argparse`, `time`, `os`)
